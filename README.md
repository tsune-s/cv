## 主な業務経歴

### Salesforceデータを用いた分析・更新パイプライン構築

**業務内容**  
SalesforceからCSV形式で抽出したデータをPythonで加工し、  
SharePoint経由でPower BIに連携する更新パイプラインを構築。

**課題**
- API権限がなく、CSV抽出が前提
- Excel・Power BI上での手作業加工が多く、属人化していた

**対応**
- Python（pandas）による前処理・集計ロジックの実装
- 加工後データをParquet形式で保存し、更新負荷を軽減
- Power BIは最終形データのみ参照する構成へ変更

**成果**
- 更新作業の標準化・属人性の低減
- データ不整合の早期検知が可能に
- 保守・改修時の影響範囲が明確化
